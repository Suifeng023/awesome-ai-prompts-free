# AI Prompt ROI Calculator Prompts

Use these free prompts to estimate whether a reusable AI prompt workflow is worth building, buying, or ignoring.

## 1. Time-saved ROI estimate

```text
You are an operations analyst evaluating whether an AI prompt workflow has positive ROI.

Workflow:
- Task name: [TASK]
- Current manual time per task: [MINUTES]
- Frequency per week: [COUNT]
- Hourly value of user's time: [USD/HOUR]
- Expected AI-assisted time per task: [MINUTES]
- Setup time to customize prompt: [MINUTES]
- Prompt/tool cost: [USD]

Calculate:
1. Weekly time saved
2. Monthly dollar value saved
3. Break-even point in days
4. Main quality risks
5. Recommendation: build / buy / ignore

Rules:
- Show formulas.
- Use conservative assumptions.
- If any input is missing, make a reasonable assumption and label it.
```

## 2. Prompt workflow prioritization matrix

```text
You are a productivity systems consultant.

I have these recurring tasks:
[PASTE TASK LIST WITH FREQUENCY, TIME COST, PAIN LEVEL, AND BUSINESS IMPACT]

Rank each task by AI prompt automation potential using:
- Repetition score: 1-5
- Context clarity score: 1-5
- Output tolerance score: 1-5
- Business value score: 1-5
- Risk score: 1-5, where 5 is high risk

Return a table with:
1. Task
2. Scores
3. Total opportunity score
4. Best prompt workflow type
5. First prompt to build
6. Whether to use human review
```

## 3. Prompt pack buying decision

```text
Act as a skeptical buyer evaluating a prompt pack.

Prompt pack:
- Name: [NAME]
- Price: [PRICE]
- Claimed use cases: [USE CASES]
- My role/business: [CONTEXT]
- Tasks I want to improve: [TASKS]

Evaluate:
1. Which use cases are immediately valuable
2. Which use cases are irrelevant
3. Estimated time saved in the first month
4. Questions to ask before buying
5. A fair price range
6. Buy / wait / skip recommendation

Be practical, not promotional.
```
