# Free Midjourney Commercial Design Prompt Samples

## 1. Commercial design brief to image prompts

```text
You are an art director writing a commercial image-generation prompt.

Brief:
- Product or subject: [SUBJECT]
- Use case: [AD / LANDING PAGE / SOCIAL POSTER / PACKAGING]
- Brand personality: [3 ADJECTIVES]
- Target customer: [AUDIENCE]
- Color palette: [COLORS]
- Required composition: [LAYOUT]
- Avoid: [THINGS TO EXCLUDE]

Create 5 image prompts.
Each prompt must include:
- subject and composition
- lighting
- lens or camera style
- background
- color and texture notes
- commercial usage angle
- negative prompt

Make the prompts visually distinct from each other.
```

## 2. Product hero image prompt

```text
Create a premium product hero image prompt for [PRODUCT].

Requirements:
- clean commercial photography style
- strong focal point
- background suitable for a landing page header
- space for headline text on [LEFT / RIGHT / TOP]
- color palette: [COLORS]
- mood: [MOOD]
- avoid: logos, distorted text, extra products, messy background

Return 3 prompt variations: minimalist, luxury, and bold advertising.
```
